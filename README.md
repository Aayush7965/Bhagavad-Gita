## Requirements
- Need an API Key from https://rapidapi.com/hub

## Installation
1. Clone and open the project.
2. Add your ```API_KEY``` that you got from RapidAPI into the index.js file.
3. Run ```npm i``` command.
4. Run ```nodemon index.js``` to run the code.
